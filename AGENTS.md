# AGENTS.md

Project rules:
- Build a generic configurable engine, not a season-specific clone.
- Use Expo + React Native + TypeScript.
- Use Supabase for online mode.
- Support online mode, local nearby mode, and single-device referee mode.
- All code, comments, docs, and UI text must be in English.
- Do not hardcode card decks or question content; use import-driven content packs.
- Never expose hider coordinates to unauthorized clients.
- Plan before implementing major features.
- Work in phases.
- Write tests for engine-critical logic.
- Keep modules small and maintainable.
